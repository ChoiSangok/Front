+ CSS 스타일 적용 우선순위

 - 선택자 우선순위
         1. 아이디 선택자
         
         2. 클래스 / 속성 / 가상 선택자
         
         3. 태그 선택자 
         
         4. 전체 선택자 
         
         
 
 - 스타일 적용방법에 따른 우선순위
 
       1. !important
       
       2. 인라인
       
       3. 임베디드
       
       4. @import
       
       5. <link>
       
       6. <link> 안에 적용된 @import ( 이건 적용하지 않는 게 좋음 )
       
       7. 브라우저의 기본 스타일
