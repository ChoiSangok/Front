+ DTD, Document Type Definition

 - DOCTYPE
 
 - 문서 유형 강의
 
 - 브라우저에게 해당 문서가 어떤 유형의 문법을 적용해서
 작성되었는지 알리는 문장
 
 - 브라우저가 DTD를 확인하고 문서를 확인하는 방법을 결정한다
 - 버전에 맞게 유효성검사(문법검사) 를 할 수 있게 된다
 
 - HTML
 
 - XML , eXtensible Markup Language
	DTD를 적용하면 문법이 적용된다

 - XHTML 1.0
 - XHTML 1.1
 
 
 
+ 태그의 기본 영역(레이아웃) 속성

 - block 요소 
          <div>, <h1> ~ <h6>, <p>, <ol>, <ul>, ...
          한 줄 전부를 차지하는 영역 속성
          
 
 - inline 요소
          <span>, <a>, <img> , ...
          
          내용물(컨텐츠)의 영역을 나타내는 속성
          
          컨텐츠를 표현하기 위한 태그로 사용된다
           
          블록(block)요소 내부에 포함시켜 작성한다
 
          ** 태그없이 사용된 단순 텍스트도 inline 요소처럼 처리한다
          
          


 + 글자 관련 태그 
 
  - <h1>, <h2>, <h3>, <h4>, <h5>, <h6>
          heading 태그
          
          제목 표현 태그
          
          h1이 큰 제목, h6로 갈수록 작은 제목
       
       
 - <p>
          paragraph 태그
          문단을 표현하는 태그
          
          공백문자( 띄어쓰기, 탭, 개행) 이 개별적으로 반영되지 않는다
          공백문자는 여러 개 적더라도 띄어쓰기 한 개로 표현한다
          
          개행 -> <br> 태그 사용
          띄어쓰기 ->&nbsp; 키워드 사용
          탭 -> 없음
          
          ** br : line bREAK
          ** nbsp : No Break SPace
          
 - <pre>
          PREformatted text
          
          html 문서에 작성되어있는 문단 형식 그대로 표현되는 태그
          
          공백문자들을 작성한 코드 그대로 표현해주는 태그
          
          스타일을 직접 지정해야하는 일이 많아져서 잘 사용하지 않는다
 
 
+ 목록

 - <ul>, <ol>, <li>
 
 - ul : Unordered List, 순서 없는 리스트
 
 - ol : Ordered List, 순서 있는 리스트
 
 - li : List Item, 리스트에 들어가는 항목
 
 - 목록태그의 자식태그로 목록태그를 넣을 수 있다
 - 중첩된 목록이 생성된다 ( 계층적으로 들여쓰기 되어 표현한다 )
 
 - CSS를 적절히 조절해서 웹화면의 메뉴를 구성할 때 사용한다
 
 + 정의 목록 태그
 
  -<dl>, <dt>, <dd>
  
  - dl : Definition(Description) List, 정의 목록
  - dt : Definition(Description) Terms, 정의 용어의 이름
  - dd : Definition(Description) Description, 정의 설명
  
  - 용어에 대한 설명, 정의 목록을 구성하는 태그
  
 
 + 하이퍼링크 태그
 
 - <a>
 
 - anchor 태그 
 
 - 클릭 시 다른 페이지로 동작하는 하이퍼링크를 용해주는 태그
 - 다른 도메인도 가능, 같은 도메인도 가능
 - 주로 텍스트에 태그를 적용하지만 이미지나 다른 태그에도 적용 가능
 
 
 - href 태그
        링크될 URL을 지정한다
 
 - target 속성
        링크된 페이지가 오픈되는 위치 조절
        
        _self - 현재 창( 기본값 )
        _blank - 새 탭
